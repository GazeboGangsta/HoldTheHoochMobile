## What shipped this session

### M4b — Missing gameplay systems (all three)

- **Score multiplier.** `+0.1× per 10s survived, capped 3×`. Applied to time-based points only (not collectibles), HUD suffixes `×1.3` when above 1×. `GameScene.multiplierFor(double elapsed)` is a testable pure function.
- **Potion spill-drain bonus.** Potion pickup grants 1 second of 4× drain rate. Implemented as transient state on `HoochBalance` (`_drainBonusRemainingSec`), triggered from `Collectible.onCollisionStart` via a `VoidCallback? onPotionBonus` plumbed through `CollectibleManager`.
- **Wobble amplitude difficulty ramp.** `HoochBalance.applyDifficulty` was a dead stub; now wired each frame with `_elapsed / difficultyRampSeconds`. Amplitude lerps 1.0× → 1.7× over 3 minutes.

Plus a fourth, necessitated by playtesting:

- **Directional drift bias.** The spec calls for "wobble (sine oscillation) + bias toward outer edges as speed increases". Only the sine half was implemented. Added `HoochBalance.setDriftDirection(±1)` + `driftRate` lerp (0.05 → 0.20 tilt/sec). Each run picks a direction at random; player counters with the tilt buttons.

### Control scheme rebuild (twice)

Initial drag-on-left-half controls collided with Android/iOS edge back-gestures. Two iterations:

1. **Tilt buttons floated in the bottom-left.** Visible, tap-and-hold, no edge collision. Worked but felt tacked-on.
2. **Dedicated bottom control strip.** Dark "underground" panel across the full width. Three buttons: ◀ tilt-left, ▶ tilt-right, ▲ jump (right-aligned). Ground visually terminates at the top of the strip; play area is above. No gestures anywhere in the play area — zero system conflicts. Multi-touch routing rewritten so lifting the tilt finger can't cut the jump short.

### Visual polish

- **Tankard rotation tied to `balance.tilt`** (`~34°` max lean). Fixes the legibility half of drift/spill — player can see which way it's leaning.
- **Dark-navy sky backdrop** matched to `bg-mountains.svg` gradient top (`#1A1A3E`) so there's no horizontal seam where the mountain layer starts.
