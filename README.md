# HoldTheHooch Mobile

Mobile companion to the browser game at [gurgles.beer](https://gurgles.beer). A 2D side-scrolling endless runner starring Gurgles the Druid Brewer Gnome — tap to jump, tilt to keep the hooch from spilling.

Built with **Flutter + Flame** for iOS and Android.

## Status

V1 in progress. Milestones M1–M3 complete (scaffold, core loop, balance mechanic). M4/M5 in flight (content + polish). Playable debug APK on Android.

See [docs/STATUS.md](docs/STATUS.md) for a live snapshot and [docs/ROADMAP.md](docs/ROADMAP.md) for the milestone plan.

## Quick start

```bash
flutter pub get
flutter run
```

Requires Flutter 3.x and the Flame 1.x game engine. See [CLAUDE.md](CLAUDE.md) for project conventions.
